-- IMDS Demo Commands

--- v1 Commands

curl top level info:		curl http://169.254.169.254/
							curl http://169.254.169.254/latest/
							curl http://169.254.169.254/latest/meta-data/
get security group:			curl http://169.254.169.254/latest/meta-data/security-groups/
get iam creds:				curl http://169.254.169.254/latest/meta-data/iam/security-credentials/EC2-ROLE 
get public keys:			curl http://169.254.169.254/latest/meta-data/public-keys/



--- v2 Commands

Saving token and curl top level:	TOKEN=`curl -X PUT "http://169.254.169.254/latest/api/token" -H "X-aws-ec2-metadata-token-ttl-seconds: 21600"` && curl -H "X-aws-ec2-metadata-token: $TOKEN" http://169.254.169.254/
Get latest list:					curl -H "X-aws-ec2-metadata-token: $TOKEN" http://169.254.169.254/latest/meta-data/    
Get Public Keys: 					curl -H "X-aws-ec2-metadata-token: $TOKEN" http://169.254.169.254/latest/meta-data/public-keys/
